# Dice with Rolls

[![Build Status](https://travis-ci.org/GPurgatorio/dice-with-rolls.svg?branch=master)](https://travis-ci.org/GPurgatorio/dice-with-rolls)
[![Coverage Status](https://coveralls.io/repos/github/GPurgatorio/dice-with-rolls/badge.svg?branch=master)](https://coveralls.io/github/GPurgatorio/dice-with-rolls?branch=master)

**"Dice with Rolls"** is a group homework from the "ICT Solutions Architect"'s course "Advanced Software Engineering", UniPi, developed by ___Dice-Tellers___.
It's a web application with which you can play around like in a social media, where users can write and react posts following randomly generated keywords.


### Setup

Simply run script.sh

```sh
$ sudo chmod +x script.sh
$ ./script.sh
```

#### N.B. You must have installed

 - `pip`
 - `python3`
 - `redis-server`
 - `celery`
